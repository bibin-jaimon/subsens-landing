# Screenshots

Add PNG, JPG, or WebP images here. They appear automatically in the Screenshots section.

- **Naming:** Use `S1.png`, `S2.png`, etc. (sorted alphabetically/numerically)
- **Format:** PNG, JPG, JPEG, WebP
- **Aspect:** 9:19.5 (phone portrait) recommended
- **Labels:** Custom labels in `vite-plugins/screenshots.ts` → `KNOWN_LABELS`. New files default to the filename (e.g. S6.png → "S6").
